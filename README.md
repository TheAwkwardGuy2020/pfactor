# pFactor

### A simple tool to return prime factors of a given number
### Install:
```sh
npm install pfactor
```
### Usage:

```js
const pFactor = require('pfactor');

console.log(pFactor(12));
```
This will log the prime factors as an array (in ascending order):
`[2,2,3]`
